# 🗳️ Class Representative Voting Form

A simple **Class Representative (CR) Voting Form** built using **HTML5**.

This project is created as part of my **Web Development Journey**, where I'm learning web development from scratch by building real-world projects.

> 🚀 This project focuses only on HTML. No CSS, JavaScript, or backend is used.

---

## 📌 Features

- 📝 Student Information Form
- 👤 Candidate Selection
- 📧 Basic Form Fields
- ☑️ Radio Buttons & Checkboxes
- 📋 Dropdown Menu
- 📤 Submit Button
- 🔄 Reset Button
- 📄 Semantic HTML Structure
